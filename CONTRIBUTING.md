
Contributing to TIME Coin Mobile
Development Setup
Android
Install Android Studio
Clone repository
Open android/ directory in Android Studio
Install dependencies (Gradle sync)
Run on emulator
Code Style
Follow Kotlin coding conventions
Use ktlint for formatting
Write unit tests for business logic
Write UI tests for critical flows
Testing
Before Submitting PR
[ ] All tests pass
[ ] No lint errors
[ ] Code is formatted
[ ] Documentation updated
[ ] Security review (if touching crypto/keys)
Pull Request Process
Fork the repository
Create feature branch (git checkout -b feature/amazing-feature)
Commit changes (git commit -m 'Add amazing feature')
Push to branch (git push origin feature/amazing-feature)
Open Pull Request
Security
Report security vulnerabilities privately to: security@time-coin.io

Do NOT open public issues for security bugs.
